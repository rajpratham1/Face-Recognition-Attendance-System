User Manual

Role-Based Quick Guide

Get Started
Student Flow

Use this path for fastest attendance marking.

  1. Register as student and scan face.
  2. Wait for teacher to open live session.
  3. Open Mark Attendance and submit.
Teacher Flow

Manage attendance windows in real time.

  1. Create course and enroll students.
  2. Start class session with duration.
  3. Close session and review report.
Admin Flow

Oversee users, compliance, and operations.

  1. Review attendance and user states.
  2. Clear face data if re-scan needed.
  3. Use print/export for records.
