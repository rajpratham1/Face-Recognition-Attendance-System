💡 How to Fix Common Issues
❌ Face Verification Failed
  • Ensure good lighting
  • Look directly at camera
  • Remove glasses/mask if possible
  • Keep face within frame
  • Don't use photos or videos
📍 Outside Classroom Radius
  • Go to the actual classroom
  • Enable location services
  • Wait for GPS to stabilize
  • Check if you're in correct room
  • Contact teacher if location is wrong
⚠️ Possible Spoofing Attempt
  • Don't use photos of yourself
  • Don't use videos or screens
  • Use live camera only
  • Blink naturally during scan
  • Move head slightly if prompted
⏰ Session Inactive
  • Mark attendance during class time
  • Check session end time
  • Don't wait until last minute
  • Contact teacher if session closed early
