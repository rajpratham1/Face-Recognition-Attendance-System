📋 CSV Format for Bulk Enrollment
Required Columns:
college_id,name,email,department,section,year,semester,course_code
Example Data:
2021001,Rahul Kumar,rahul@example.com,CSE,A,3,5,CS301
2021002,Priya Sharma,priya@example.com,CSE,A,3,5,CS301
2021003,Amit Singh,amit@example.com,CSE,B,3,5,CS302
2021004,Neha Gupta,neha@example.com,ECE,A,2,3,EC201
Rules:
  • Header row is required
  • Email must be unique
  • If student exists, only enrollment is created
  • If student doesn't exist, created with password = college_id
  • Course must exist (create courses first)
  • Section must match student's section
💡 Tip: Students will receive their college_id as default password. Ask them to change it after first login.
